03 / Method

Evidence first. Decisions second.

The LLM synthesizes fetched evidence; deterministic checks own schema, citations, and catalog overlap.

  1. 01Coverage lookup

    Composio SDK/catalog lookup sets existing toolkit coverage.

    composio_lookup.py
  2. 02Docs research

    Official API, authentication, access, and MCP pages are collected.

    docs_research.py
  3. 03Schema synthesis

    The model returns a locked 19-field record with cited URLs.

    synthesis.py
  4. 04Source audit

    Invalid labels, weak coverage, contradictions, and invented links fail closed.

    batch_pipeline.py
  5. 05Official-doc check

    Priority apps are adjudicated against vendor docs, with every recorded miss preserved.

    handcheck.py
